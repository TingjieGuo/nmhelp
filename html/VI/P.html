NONMEM Users Guide Part VI - PREDPP

Preface. to 4th Edition

The appearance of this 4th edition of the PREDPP Guide corresponds to the release of PREDPP 7.4.0 and NONMEM 7.4.0. This is the first new edition since PREDPP Version III and NONMEM IV in 1992.

Significant changes since the previous version are marked with bars.

Versions of PREDPP are now numbered to match those of NONMEM. In the following, new features are arranged by the versions in which they were introduced.

PREDPP. IV (NONMEM V)

ADVAN11, ADVAN12, TRANS6

New ADVAN routines. These implement the analytic solution for the Three Compartment Linear Model. New TRANS routine. See Chapter VII, Sections C.11, C.12.

$MODEL

A user-written MODEL subroutine is no longer requrired. NM-TRAN can generate the desired subroutine. See Chapter VI, Section B.

Endogenous drug

A record with AMT=0, RATE=0, SS>0, and II=0 allows steady-state amounts to be computed by PREDPP based on endogenous input given by the DES routine. See Chapter I, Section I.

New value of SS data item: SS=3

With Steady State routines SS6 and SS9 and SS13, the existing state vector (compartment amounts and eta derivatives) may be used as the initial estimate in the computation of the steady-state amounts. See Chapter V, Section F.2.

Models with no Basic PK parameters

With ADVAN routines 6, 8, 9, and 13, MODEL may now define models with no basic PK parameters. See Chapter IV, Section B.

ADVAN9 as an Algebraic Equation Solver

ADVAN9 may be used when the system consists of only algebraic equations (i.e., no differential equations). See Chapter VI Section B.

Changes to DES and AES Routines

Argument DT

With subroutine DES, a new argument DT must be declared. The partial derivative of DADT(i) with respect to T must be stored in DT(i). See Chapter VI, Section C.

New Structure for DA, DP, DT: Full vs. Compact

See Appendix IV.

AES and DES Initialization at ICALL=1

AES and DES may return information to PREDPP at the initialization call. See Chapter VI, Sections C.B and E.B.

Display AES and DES Defined items

DES- and AES-defined items may be displayed in tables or scatterplots and are computed at the event time in the data record. See Chapter VI, Sections C.C and E.C.

Theta and Data record items in DES and AES

In DES and AES: Theta and EVTREC are usable directly. See Chapter VI, Sections C.A and E.A.

PK, ERROR, TRANS: NACTIV,IDXETA

A user written PK, TRANS, or ERROR routine may compute derivatives for all ’s, or optionally, to reduce run time, only for active ’s. See Chapter III, Section E.5.

ERROR: IDEF

At ICALL=1, a value may optionally be stored in IDEF to reduce run time. It tells if ERROR uses derivatives of state vector A (see below for similar feature of PK). See Chapter IV, Section D.A.

PK: Calling protocol

Substantial time savings are possible when the PK subroutine limits calls to once per individual record (CALLFL=1 or ONCE PER IR). This is especialy important with ADVAN5 and ADVAN7. See Chapter III, Section H.

INFN: PASSRC, NEWND

PASSRC contains the data record at ICALL values 0, 1 and 3. PASSRC changes in conjunction with calls to PASS, at which time transgeneration is permitted. NWIND contains the NEWIND value at ICALL 0, 1 and 3. NWIND changes value in conjunction with calls to PASS. See Chapter VI, Section A.

$BIND

This is an NM-TRAN feature useful for PREDPP. It specifies which values from which event records are used in PK subroutines at non-event times. See Chapter III, Section I.A.

PREDPP. V (NONMEM VI)

MTIME

Model Event Time feature. See Chapter III and Appendix III.

A_0,

A_0FLG

Compartment Initialization feature. See Chapter III, Section I.B.

I_SS

feature

Initial Steady State feature. See Chapter III, Section I.B and Chapter V, Section F.3.

New

value of CALL data item

CALL may be 10 to force a call to ADVAN9 See Chapter IV, Section V.J.

New

values of CMT data items.

CMT data item values 100 and 1000 may be used to indicate the default compartment for output. See Chapter IV, Section V.H.

PK:

State vector

Like ERROR, PK may use elements of the state vector (compartment amounts A) on the right. TSTATE (time of state vector A) may be used on the right. See Chapter III, Section I.A.

PK:

IDEF

At ICALL=1, a value may optionally be stored in IDEF to reduce run time. It tells if PK uses derivatives of state vector A. See Chapter III, Section G.

PK,

ERROR: New values of ICALL argument

New values 5 and 6 of ICALL indicate that NONMEM is computing raw data averages or expectations. PK and ERROR routines may have to provide code for these calls. See Chapter III, Section C and Chapter IV, Section B.2.

Changes to SUBROUTINE TOL. See Chapter VI, Section D.B.

New

data items XVID XVID1 XVID2 XVID3 XVID4 XVID5

"Repeated Observation Records" feature exists to assist in specialized methodologies such as stochastic differential equations. See Chapter V, Section B.

Larger models

The default numbers of compartments and parameters are larger. Even larger values can be requested using options PC and PG of the $SIZES record. Other options such as MAXFCN affect PREDPP. For example, $SIZES MAXFCN=9000000 may be used with General Non-Linear models (ADVAN6, ADVAN8, ADVAN9, ADVAN13) to request more function evaluations than the default value. See Chapter VII, Section A.
